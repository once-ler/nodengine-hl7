ZSN Segment

A new custom segment (ZSN) is added to the outbound Epic MDM^T08 template that contains information for the chart search binary data change notification. The ZSN segment must follow the OBX segment in the MDM^T08 template and only contains one value needed for the notification to Epic.

Field Description Static Value Example Value Additional Information
1 Specialty     Not Used
2 Subspecialty     Not Used
3 Document Status    

Not Used

When ZSN-3 is not received, the scan status is retained (for update messages), and is defaulted based on system settings.

4 Transaction ID     Not Used
5 Binary Data Update Instant $$TIMESTAMP 20230119064221

This field contains the date and time the binary data for the scan or image was last updated.

This date and time is evaluated for external scans, such as those from a document management system.

Epic compares the ZSN-5 date with the date that is already stored in Epic. If the ZSN-5 date is newer, Epic then uses the Binary Data Retrieval API to get the latest document binary data.

The $$TIMESTAMP value contains the date and time the HL7 message was sent.
